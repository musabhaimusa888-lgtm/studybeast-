# 🧠 StudyBeast — AI Study Assistant

A powerful AI-powered study companion for college students. Generate quizzes, get assignment help, or create stunning presentations — all in seconds.

![StudyBeast](https://img.shields.io/badge/StudyBeast-AI%20Study%20Assistant-22c55e?style=for-the-badge&logo=brain&logoColor=white)

## ✨ Features

### 📝 Quiz Generator
- Generate multiple-choice quizzes on 6+ subjects
- Instant feedback with detailed explanations
- Adjustable difficulty levels (Easy/Medium/Hard/Mixed)
- Score tracking and performance analytics

### 📄 Assignment Helper
- Structured outlines for essays, research papers & presentations
- Key objectives and talking points
- Reference suggestions by source type
- Copy-to-clipboard for easy export

### 🎨 Presentation Generator *(NEW)*
- Beautiful slide decks with AI-matched stock images
- 8 unique slide types (Title, Content, Stats, Quote, Comparison, etc.)
- 4 visual themes (Modern, Academic, Creative, Minimal)
- Fullscreen mode, keyboard navigation, grid view
- 56 curated images across 8 topic categories

## 🚀 Tech Stack

- **React 19** + **TypeScript**
- **Vite** for blazing-fast builds
- **Tailwind CSS 4** for styling
- **Framer Motion** for smooth animations
- **Lucide React** for icons

## 🛠️ Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📦 Deployment

This project is optimized for **Vercel** deployment:

1. Push to GitHub
2. Import project in Vercel
3. Deploy automatically

No configuration needed — Vite is auto-detected.
